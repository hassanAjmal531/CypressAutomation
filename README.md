# CypressAutomation



**Project Setup**
Run "npm install" 
Run "npx cypress open"
configure E2E testing from cypress UI

**Run Tests**
Run "npx cypress open"
select "DemoQaTests.cy.test" to run tests

**Base Url** = "https://demoqa.com/"

**Description**
Tests are automated using the page object model using Cypress.io.
